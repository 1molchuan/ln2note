# $\ln 2 < 0.7$ 的估值与探索

> “过去更重要。” —— 一篇陪伴我一年的文章。


## 📖 项目简介

这是一个关于数学常数估值的探索性笔记。项目起源于高中时期的一份手写笔记，旨在通过多种数学方法证明并探索 $\ln 2 < 0.7$ 这一命题。

从最初的纸笔推导，到 LaTeX 排版，最终通过 Quarto 构建为现代化的学术网页。

**在线阅读地址：** [https://1molchuan.github.io/ln2note/](https://1molchuan.github.io/ln2note/)

## 🧮 核心内容

本项目涵盖了从初等数学到高等数学的多种估值方法：

* **切线放缩**：基础的凹凸性分析。
* **积分构造**：通过构造函数与积分变换进行逼近。
* **根式加强**：引入根号进行非线性拟合。
* **级数展开**：泰勒展开 (Taylor Series)、帕德近似 (Padé Approximant) 与连分式。
* **数值积分**：梯形法则与辛普森公式。

## 🛠️ 技术栈

本项目采用“代码化写作”流程：

* **核心内容**：LaTeX (MathJax)
* **构建工具**：[Quarto](https://quarto.org/)
* **部署托管**：GitHub Pages

## 🚀 本地运行

如果你想在本地查看或修改源码：

1.  克隆仓库：
    ```bash
    git clone [https://github.com/1molchuan/ln2note.git](https://github.com/1molchuan/ln2note.git)
    ```

    ```bash

    ```
